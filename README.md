# Larave Livewire Quiz System
### Clone repo

- `git clone git@github.com:dung-nguyentrung/quiz.git`
- `cd quiz`
- `cp .env.example .env`
- Tạo CSDL và đổi tên DB_DATABASE trong env
- `composer install`
- `npm install`
- `npm run dev`
- `php artisan key:generate`
- `php artisan migrate`
